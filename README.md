# Investment-Checklist
#### 投资检查清单

佛说"一念万相",人的念想是千变万化的,人的念想也是最不可靠的,在投资的过程中,最难的就是排除外界的干扰,用逻辑和辨证控制自己的念想,避免自己的操作受到外界环境的影响,甚至使投资产生亏损.所以,基于这个目的,制作了这个投资检查清单,让自己在每次心态动摇的时候对照一下,及时收回心神,抓住主要矛盾.

事物是不可能跳跃式发展的,企业的发展也是由量变引起质变的过程,是一个要经过时间发酵不断变化的过程,所以慢和等待是投资过程的必须也是关键.慢:就是不要着急,不要觉得现在不投入就错过了机会,不要觉得别人赚了大钱,自己就应该赚,甚至亏损都是一种耻辱.等待:就是要等宏观和企业的各种情况变得匹配合理,等待股票的价格到达了那个合理的区间.还有就是在慢和等待的过程中,跟踪企业的情况,抓住企业生产产品的供需变化这个核心的矛盾点,再扩展到企业的运营,把投资的企业当成自己的企业,用实事求是的态度,对企业的各种情况掌握得非常透彻.

难就难在认真,难就难在深耕,难就难在用实事求是的态度对待.懒惰和欺骗自己是人们经常做的,但在投资的过程中,懒惰和欺骗自己就是在与赚钱背道而驰,所以要从心底反复叩问自己:"了解清楚了没有""计算好了没有""核实对了没有",坚决摒弃是是而非,粗枝大叶,"认为是这样","应该是这样"等等,这种模糊的,不负责任的做事方式.

成功是干出来的,赚钱是做出来的,实践是检验真理的唯一标准,`实践->认识->再实践->再认识`,从实际的投资生涯中,不断提升自己的认知,做到精通和专业,这才是目标.

## 投资前的检查项目

### 企业战略的检查
1.公司的产能扩张路径是否清晰可见,是否具有极高的确定性.
2.买入时企业生产的产品(油气\材料\金属等周期)的价格是否处于低位.
3.对公司的护城河是否误解.
4.行业的渗透率,企业的市场占有率

### 企业财报的检查
1.公司的杠杆率是否太高.
2.买入时公司的估值是否处于显著低估状态(拥有安全边际).
3.企业的未来现金流折现

## 避免的内容
1.股价的短期波动.
2.市场的热点和题材炒作.
